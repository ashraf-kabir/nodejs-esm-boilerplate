```
node-es6-boilerplate/
├── src/
│   ├── config/

│   │   └── index.js

│   ├── app.js
│   └── server.js
├── .env.dist
├── .gitignore
├── initialize.sh
├── package.json
└── README.md
```

## Installation
1. Clone the repository
2. Run `./initialize.sh` to install dependencies and create `.env` file
3. start the dev server using `npm run dev`
4. Visit `http://localhost:5000` to see the server running
